[hyprwayland-scanner](https://github.com/hyprwm/hyprwayland-scanner), also called hw-s, is a utility to generate sources and headers for wayland protocol specifications.

It is similar to wayland-scanner, but instead of C, it generates neat and safe C++ implementations.
